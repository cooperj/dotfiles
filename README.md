# dotfiles

## Install

0. Install Zsh

```
sudo apt install zsh
```

1. Set Zsh as the default shell environment

```
chsh -s $(which zsh)
```

2. (Reopen Shell Session and then) Install "Oh My Zsh"

```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
